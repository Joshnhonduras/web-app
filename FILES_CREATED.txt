WHAT EACH FILE DOES
════════════════════════════════════════════════════════════════════════════

usageService.ts
  Core Functions:
    • getUsageData() - Get usage from localStorage
    • initializeTrial() - Start trial on first message
    • recordMessageUsage(length) - Track user message
    • recordResponseTokens(count) - Track API response
    • getRemainingTokens() - Get tokens left
    • isTrialExhausted() - Check if over limit
    • getEstimatedWords(tokens) - Convert to words
    • getUsageSummary() - Get full report
    • resetUsageData() - Clear for testing

freeTierClient.ts
  Core Functions:
    • sendMessageFreeTier(messages, systemPrompt) - Send to backend
    • checkFreeTierHealth() - Check if service up
    • getFreeTierModels() - List available models

disclaimers.ts
  Core Functions:
    • getDisclaimerAcknowledgment() - Check if user acknowledged
    • acknowledgeDisclaimers() - Mark as acknowledged
    • getFreeTierDisclaimers() - Get disclaimers for trial
    • getPaidApiDisclaimers() - Get disclaimers for paid users

TrialWelcomeModal.tsx
  Shows:
    • 1,000 token trial explanation
    • Approximate word count
    • Option: Get API key with setup guides
    • Option: Subscribe $9.99/month
    • Step-by-step instructions for Groq/OpenRouter/OpenAI

DisclaimerBanner.tsx
  Shows:

    • Expandable details

    • Must be checked to proceed
