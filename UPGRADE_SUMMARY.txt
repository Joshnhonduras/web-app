SUPPORT & QUESTIONS
================================================================================

All CSS changes use semantic custom properties (CSS variables) for easy
maintenance. To modify colors, spacing, or animations:

1. Edit variables in src/index.css (:root section)
2. Changes automatically apply throughout the app
3. No component code needs to be updated

Example: Change primary color globally
  Before:  var(--primary-600) = #4f46e5
  After:   var(--primary-600) = #your-color
  Result:  All buttons, headers, accents update automatically
